# 🚀 Free Resume Hiring Platform

A full-stack web application that connects **job seekers** and **recruiters**. Users can upload resumes, and recruiters can browse and shortlist candidates efficiently.

---

## ✨ Features

* 🔐 User Authentication (Login / Signup)
* 📄 Resume Upload System
* 👨‍💼 Recruiter Dashboard
* 🔍 View & Filter Candidates
* 📱 Fully Responsive UI
* ⚡ Fast and simple user experience

---

## 🛠️ Tech Stack

* **Frontend:** React.js, HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Deployment:** Vercel (Frontend), Render (Backend)

---

## ⚙️ Available Scripts

In the project directory, you can run:

* `npm start` → Runs the app in development mode
* `npm run build` → Builds the app for production
* `npm test` → Runs tests
